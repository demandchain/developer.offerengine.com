User Access Tokens

Access tokens are temporary tokens which allow users to call APIs directly. Think JavaScript checkout flows, and the like.

Create an access token

POST /users/:user/access_tokens

Parameters

expires_in
Integer time to live in seconds
scope
  • String comma separated resources Examples: user, credit_card
  • all for unlimited
access
String read, write, read,write

Response

Status: 200 OK
{
  "status": "success",
  "access_token": {
    "id": "zzzaaaa123",
    "user_id": "abc123",
    "expires_in": 600,
    "scope": "user,credit_card",
    "access": "read,write"
  }
}
