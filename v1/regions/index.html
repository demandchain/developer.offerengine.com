Regions

  • Get a region
  • List all regions

Get a region

GET /api/v1/regions/:region_id

Response

Status: 200 OK
{
  "status": "success",
  "region": {
    "id": "atlanta",
    "active": true,
    "name": "Atlanta"
  }
}

List all regions

Returns an array of region representations.

GET /api/v1/regions

Response

Status: 200 OK
{
  "status": "success",
  "regions": [
    {
      "id": "atlanta",
      "active": true,
      "name": "Atlanta"
    }
  ]
}
